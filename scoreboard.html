Custom Scoreboard Mod

SkyHanni

How to use:

  1. Click the "Choose File" button and select a .png image.
  2. Click the "Create Texturepack" button.
  3. Download the generated .zip file.
  4. Place the .zip file in the "resourcepacks" folder of your Minecraft directory.
  5. Enable the resource pack in the game.
  6. Enable the option to use the background in the SkyHanni config (CustomScoreboard config if Modern).
